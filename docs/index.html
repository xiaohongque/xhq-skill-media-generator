What your assistant can create for your shop

Under the hood, each capability is a routing-group key passed to one REST API. The backend picks the best available generator — you only say what you want.

  • video.seedance-2.0 — Seedance 2.0
  • video.seedance-fast-2.0 — fast tier
  • video.seedance-mini-2.0 — mini tier
  • video.grok-image-1.5 — Grok Image 1.5

How it works

Three steps to your first asset

Built for sellers, not just developers. If you can install a skill into your AI assistant, you can run the whole studio.

1

Get a free API Key

Log into xiaohongque.com → Settings → Account → API Keys → “+ Create Key”. Copy the one-time sk_… key.

2

Drop the skill into your agent

Place the skill folder under .codebuddy/skills/ and set XHQ_API_KEY. Or just ask your agent to install it for you.

3

Tell it what to make

Show a product photo and describe the result. It submits, generates, and hands you OSS URLs — 雀豆 deducted automatically.
