How it works

From zero to first render in 3 steps

1

Create an API Key

Log into xiaohongque.com → Settings → Account → API Keys → “+ Create Key”. Copy the one-time sk_… key.

2

Install the skill

Drop the skill folder under .codebuddy/skills/ (or ~/.codebuddy/skills/) and set XHQ_API_KEY. Or reference SKILL.md from any AGENTS.md.

3

Let the agent generate

Ask your agent to make a video / image / digital human. It submits, polls, and returns OSS URLs — 雀豆 deducted automatically.

Quickstart

Run your first task

scripts/run_task.js does submit + polling in one shot. Node 18+, zero dependencies.

bash
export XHQ_API_KEY="sk_..."                # the key you created
# XHQ_API_BASE is optional; defaults to https://app.xiaohongque.com

node scripts/run_task.js \
  --provider video.seedance-2.0 \
  --params '{"prompt":"A calm ocean wave rolling toward shore","image_urls":["https://.../ref.jpg"],"duration":5,"aspect_ratio":"16:9"}'

🔐 Security & billing

  • A key equals your account — never share it or commit it to a repo.
  • Revoke anytime from the key list; revoke then recreate if you suspect a leak.
  • Least privilege — default scope is only generate + task-query.
  • Billing needs no setup: 雀豆 are deducted per task and appear in 雀豆明细 automatically.
